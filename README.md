## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory with pages and global styles
- `/components` - React components (Hero, MemberCard)
- `/public` - Static assets (logo, images)

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## Customization

You can customize the band members in `app/page.tsx` by editing the `bandMembers` array.
